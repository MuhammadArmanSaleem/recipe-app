Design a premium mobile-first AI recipe transformation app with a warm, minimal, modern cooking aesthetic.

IMPORTANT:
This is NOT a recipe browsing app like Pinterest.
This is an AI-powered recipe workspace where users paste recipe video URLs and instantly transform recipes dynamically.

The app should feel like:
- an intelligent kitchen workstation
- calm and premium
- modern but warm
- highly interactive
- optimized for one-hand mobile usage

STYLE:
- Warm editorial food aesthetic
- Cozy luxury café vibe
- Minimal clutter
- Soft rounded UI
- Smooth animations
- Elegant spacing
- Calm and focused
- Premium tactile interactions

AVOID:
- Generic SaaS dashboard design
- Cold AI blue colors
- Cyberpunk aesthetics
- Too many cards
- Busy layouts
- Social-media-feed UI

COLOR PALETTE:
- Cream: #F5E9DA
- Soft Beige: #E7D7C9
- Warm Brown: #5C3B2E
- Coffee Dark Brown: #3A261D
- Terracotta Accent: #C47A5A
- Optional Olive Accent: #7A8B5A

TYPOGRAPHY:
- Elegant serif headings similar to Playfair Display or Cormorant Garamond
- Clean modern sans-serif body text similar to Inter or Manrope
- Large typography
- Lots of whitespace

DESIGN SYSTEM:
- rounded-3xl cards
- floating UI elements
- subtle shadows
- soft gradients
- smooth segmented controls
- tactile buttons
- premium animations
- glassmorphism only subtly
- warm dark mode using brown tones instead of black

MAIN UX CONCEPT:
The app is centered around ONE primary workspace screen.

USER FLOW:
1. User opens app
2. Immediately sees URL input
3. User pastes TikTok / Instagram / YouTube recipe URL
4. AI extracts recipe
5. Recipe and ingredients appear dynamically on SAME screen
6. User enters AI prompts to modify recipe
7. Recipe + ingredients update live
8. User can order missing ingredients

MAIN SCREEN LAYOUT:

--------------------------------
HEADER
--------------------------------

Minimal top area:
- Small logo
- Profile icon
- No traditional navbar

--------------------------------
HERO TEXT
--------------------------------

Elegant heading:
“Transform Any Recipe Video Into Your Perfect Meal”

Minimal supporting text.

--------------------------------
URL INPUT SECTION
--------------------------------

Large premium URL input field.

Placeholder:
“Paste TikTok, Reel, or YouTube recipe URL”

Large CTA button:
“Extract Recipe”

This is the primary focus of the app.

--------------------------------
AI MODIFICATION BAR
--------------------------------

IMPORTANT:
This sits directly BELOW the URL input and ABOVE all recipe content.

It controls everything dynamically.

Design it like a floating premium AI command bar.

Placeholder examples:
- “Make this high protein”
- “Reduce calories”
- “Bulk version”
- “Make it keto”

Include:
- AI sparkle icon
- send button
- premium glow/focus state

--------------------------------
RECIPE / INGREDIENT SELECTOR
--------------------------------

Below the AI bar create TWO LARGE segmented pill selectors side-by-side:

[ Recipe ]    [ Ingredients ]

IMPORTANT:
- These are NOT tiny tabs
- They should feel tactile and smooth
- iOS-style segmented controls
- Large touch targets
- Smooth sliding transition animation

--------------------------------
RECIPE VIEW
--------------------------------

When “Recipe” is selected show:

- Large food image
- Recipe title
- Calories
- Protein
- Prep time
- Goal tags
- Beautiful cooking instructions
- Elegant typography
- Scrollable content
- Clean spacing

--------------------------------
INGREDIENT VIEW
--------------------------------

When “Ingredients” is selected show:

- Ingredient checklist UI
- Large clean ingredient rows
- Smooth checkbox interactions

Example:
☑ Chicken Breast
☐ Greek Yogurt
☑ Garlic

At the bottom place a sticky/floating CTA button:

[ Get Missing Ingredients ]

IMPORTANT:
This button opens a separate grocery selection screen.

--------------------------------
GROCERY SELECTION SCREEN
--------------------------------

This is a separate page/modal/sheet.

Users can:
- select ingredients they already have
- unselect ingredients they don’t need
- choose what to order

Design:
- clean grocery checklist
- premium minimal aesthetic
- warm colors
- soft floating cards

Bottom CTA:
[ Order Selected Ingredients ]

IMPORTANT:
If the recipe gets modified by AI, this grocery list updates dynamically too.

--------------------------------
LIVE AI FEEDBACK
--------------------------------

When AI modifies recipe show subtle animated states like:

“Updating Recipe...”
“Recalculating Macros...”
“Updating Ingredients...”

This should feel intelligent and alive.

--------------------------------
BOTTOM NAVIGATION
--------------------------------

DO NOT use traditional top navigation.

Use a floating bottom pill navigation optimized for one-hand usage.

STYLE:
- floating rounded-full container
- translucent cream/beige background
- subtle blur
- soft shadows
- premium iOS feel

Example layout:

[ Workspace ]   [ Saved ]   [ Profile ]

OR

[ Saved ]   [ ✨ ]   [ Profile ]

The center AI button should feel special and elevated.

--------------------------------
ANIMATIONS
--------------------------------

Use:
- Framer Motion style transitions
- smooth easing
- floating interactions
- soft fades
- tactile presses
- sliding segmented controls
- premium mobile interactions

The app should feel smooth, calm, and intelligent.

--------------------------------
OVERALL FEEL
--------------------------------

The final UI should feel like:
- Apple-level polish
- modern AI kitchen companion
- premium cooking tool
- warm intelligent workspace

NOT:
- a social feed
- a dashboard
- a recipe marketplace

The app should prioritize:
- simplicity
- speed
- focus
- elegant interactions
- warm premium aesthetics
